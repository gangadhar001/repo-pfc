\section{Fase de Transición} \label{sec:faseTransicion}

En esta fase, compuesta de una única iteración, es donde el producto software se prepara para su entrega al cliente, incluyendo los últimos detalles de implementación y pruebas.

\subsection{Iteración 9}
 
Al alcanzar esta última iteración, se han realizado las últimas pruebas globales para comprobar el correcto funcionamiento del sistema y se han corregido pequeños \textit{bugs} detectados.

A continuación, se crea la distribución del sistema para la entrega al cliente, contando en este caso con dos componentes, referentes al servidor y al cliente. Para ello, se crean los paquetes JAR (\textbf{J}ava \textbf{AR}chive) ejecutables, tanto para el servidor, llamado \textit{DMPTool-Server.jar}, y para el cliente, llamado \textit{DPMTool-Client.jar}.

Para finalizar, se realiza la documentación del producto software y se redactan los manuales de configuración, instalación y manuales de usuario del sistema. Dichos manuales se han incorporado al presente documento como anexos.
