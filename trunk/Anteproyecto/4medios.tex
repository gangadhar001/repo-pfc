\section{Medios que se pretenden utilizar}

Para el desarrollo del PFC se utilizarán una serie de medios software, tanto para la especificación y definición del sistema, como para su implementación. Dichos medios son: 

\begin{milista}
	\item \textbf{Eclipse} \cite{eclipse} (\textit{versión 3.6 Helios}), ya que la herramienta a desarrollar es un plug-in para dicho IDE.
	\item \textbf{Java} \cite{java}, en su última versión (\textit{versión 6 Update 23}).
	\item \textbf{Apache Commons Configuration} \cite{apacheCommons}, para leer archivos de configuración de diferentes tipos (\textit{properties}, \textit{xml}, etcétera).
	\item \textbf{JAXB (Java Architecture for XML Binding)} \cite{jaxb}, para poder mapear y exportar clases Java a un fichero XML.
	\item \textbf{Zest: The Eclipse Visualization Toolkit} \cite{zest}, para visualizar el conocimiento en forma de grafo o árbol.
	\item \textbf{JFreeChart} \cite{jfreechart}, para mostrar gráficos.
	\item \textbf{MySQL} \cite{mysql}, para la elaboración de la base de datos de conocimiento con la que debe comunicarse la herramienta. 
	\item \textbf{Visual Paradigm} \cite{visualparadigm}, para la realización de los diferentes diagramas de la herramienta. 
	\item \textbf{LaTeX} \cite{latex}, para la elaboración de la documentación. 
\end{milista}
