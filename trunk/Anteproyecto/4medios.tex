\section{Medios que se pretenden utilizar}

En este apartado se describen los medios tanto hardware como software necesarios para
el desarrollo del proyecto.

\subsection{Medios Hardware}

Se utilizará un ordenador de sobremesa o un portátil para llevar a cabo el desarrollo de la herramienta.

\subsection{Medios Software}

La herramienta se desarrollará sobre un ordenador con sistema operativo Windows XP o Windows 7. 
Otro software que será necesario es el siguiente: 

\begin{milista}
	\item Eclipse \cite{eclipse} (\textit{versión 3.5 Galileo}), ya que la herramienta a desarrollar es un plug-in para dicho IDE.
	\item Java \cite{java}, en su última versión (\textit{versión 6 Update 22}).
	\item Visual Paradigm \cite{visualparadigm}, para la realización de los diferentes diagramas de la herramienta. 
	\item MySQL \cite{mysql}, para la elaboración de la base de datos de conocimiento con la que debe comunicarse la herramienta. 
	\item LaTeX \cite{latex}, para la elaboración de la documentación. 
\end{milista}
